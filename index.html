Introduction To HTML

HTML stands for:
HyperText
Markup
Language
1.HTML elements are the building blocks of HTML pages.
2.HTML elements are described by tags.
3.HTML describes the structure of web pages using mark up.

This is an example of a simple HTML document.
< !DOCTYPE html >
< head > < /head >
< body >
< h1 > Heading 1 < /h1>
< h2 >Heading 2 < /h 2>
< p > paragraph < / p >
< / body >
< / html >
HTML tags normally come in pairs
For Example :
< p > < / p >
< b > < / b >
< body > < / body >
< head > < / head >
These are called tags.
Notice that the end tag has a forward slash right before the letter?
That tells the browser that this is a closing tag.
The < DOCTYPE html >simply tells the browser that this is a html5 document.
The < head >tag is where data or information about the document is placed.
The < h1 >tag tells the browser that this is a first level heading.
The < h2 >tag tells the browser that this is a second level heading.
